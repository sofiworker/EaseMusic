# EaseMusic
一个自用的网易云音乐第三方App

Ps：目前正处于开发阶段
# 起因
由于网易云加入的广告越来越多，然后偶然之间看到了[NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi)这个库能通过跨站请求伪造获取网易云的Api，所有就有想法写个自用的清洁、干净的App，也算是以此项目对Android学习的总结。
